# Poverty-Headcount-Ratio-Predictor
Machine Learning model to predict poverty headcount ratio using socio-economic indicators (World Bank dataset).

## Project Overview
This project develops a **Machine Learning model to predict the Poverty Headcount Ratio** using socio-economic indicators obtained from the **World Bank dataset**.
The goal of the project is to analyze economic indicators and estimate poverty levels using data-driven machine learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset
The dataset used in this project is obtained from **World Bank Open Data** and contains socio-economic indicators such as:
* GDP per capita
* Employment statistics
* Education indicators
* Economic development metrics
These features are used to predict the **poverty headcount ratio**.

## Machine Learning Workflow
1. Data Collection (World Bank Dataset)
2. Data Cleaning and Preprocessing
3. Feature Selection and Engineering
4. Model Training
5. Model Evaluation
6. Poverty Ratio Prediction

## Model Used
The following machine learning models were explored:
* Linear Regression
* Random Forest Regressor
* Gradient Boosting Regressor
These models help analyze relationships between economic indicators and poverty levels.
